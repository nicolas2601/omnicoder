# SEO Skill

Use this skill for comprehensive SEO analysis, on-page optimization, technical
SEO audits, keyword research, meta tags, schema markup, sitemap generation,
Core Web Vitals, crawlability and indexability issues. Covers Google, Bing,
Baidu ranking factors. Also handles AI search optimization (GEO) for ChatGPT,
Perplexity, and AI Overviews.

## When to use

- "analyze my site for SEO"
- "SEO audit"
- "improve search rankings"
- "meta tags"
- "structured data"
